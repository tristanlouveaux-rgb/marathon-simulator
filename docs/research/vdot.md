What VDOT represents (and why it works)

VDOT is best thought of as “performance-equivalent VO₂max”: a single number that implicitly bundles the big drivers of endurance performance (VO₂max, fractional utilization / threshold, and running economy) because it’s derived from the relationship between speed and how long you can sustain it (via the percent-of-max function). That’s why Daniels can use one number to generate coherent pace zones/tables.

This is aligned with the broader physiology literature: endurance performance is not explained by VO₂max alone; lactate/threshold markers and running economy are major independent contributors, so a performance-derived scalar tends to behave better than any single physiology metric in isolation.  

Is VDOT “running economy” or “running fitness”?

Neither, exactly.
	•	Running economy is typically defined as the oxygen/energy cost at a given speed (you can have great economy with mediocre VO₂max, and vice versa).  
	•	VDOT is a performance index that absorbs economy + VO₂max + fractional utilization into one number because it’s backed out from performance (or from Daniels’ model of it).

So your instinct is right: VDOT is not “pure fitness” in the physiological sense. It’s a useful internal state variable for a training engine because it’s tightly coupled to what users care about (race times) and it maps nicely into training paces.

Does research support using VDOT-style performance models?

Directly: Daniels’ equations are widely used in coaching practice and work well as an engineering approximation for pace prescription and time prediction. Indirectly (strongly): the determinants literature supports the idea that a single performance-calibrated index will often outperform single raw physiological signals (like watch VO₂max) when predicting race outcomes.
